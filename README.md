Overview

This experiment focuses on exploring statistical and specialised data visualisation techniques using Python. Various plots are implemented to analyse distributions, relationships, and patterns in datasets. Two datasets are used to demonstrate different visualisation approaches.

Tools and Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Objectives
To understand statistical data visualisation techniques
To identify patterns, trends, and relationships in data
To detect outliers and correlations
To represent data using both basic and advanced plots
Dataset Description
Dataset 1

Contains categorical and numerical data:

Category
Value
Sales
Profit
Dataset 2

Simulated customer dataset with:

Customer_ID
Age
Income
Loan Amount
Credit Score
Visualisations Implemented
Dataset 1
1. Area Plot
Represents variation of values across categories
Shows magnitude using filled regions
2. Pie Chart
Displays proportion of each category
Useful for percentage distribution
3. Donut Chart
Modified pie chart with central hole
Improves visual clarity
4. Boxplot
Detects outliers in the data
Shows median, quartiles, and spread
5. Heatmap
Displays correlation between variables
Helps identify relationships between Sales and Profit
6. Bubble Plot
Represents three variables simultaneously
Size and colour indicate additional dimensions
7. Seaborn Bubble Plot
Enhanced visualisation with better aesthetics
Uses size and colour encoding
Dataset 2
1. Area Plot (Income vs Loan Amount)
Compares income and loan amount trends
Data sorted by age for better interpretation
2. Pie Chart
Represents distribution of income with respect to credit score
3. Box Plot
Shows distribution and variability of Income and Loan Amount
Helps identify outliers
4. Heatmap
Displays correlation among Age, Income, Loan Amount, and Credit Score
5. Bubble Plot
Visualises relationship between Age and Income
Size represents Loan Amount
Colour represents Credit Score
Key Insights
Statistical plots provide deeper understanding of data distribution
Boxplots are effective in detecting outliers
Heatmaps reveal correlations between variables
Bubble plots allow multi-dimensional analysis
Visualisation improves interpretation of complex datasets
Conclusion

This experiment demonstrates the importance of statistical visualisation techniques in data analysis. By applying various plotting methods, meaningful insights can be derived from raw data. These techniques are essential for analysing real-world datasets and supporting data-driven decisions.
